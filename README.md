# vapeur
Projet vapeur MSCA
Ce projet permet de gerer une liste de jeux, ajouter des jeux, leurs mettre des éditeurs, des genres. il permets de trier les jeux par genre et ou par editeur.

# cloner le depot git
cloner le dossier sur son ordinateur
# lancer le serveur
node app.js
le serveur sera lancé sur http://localhost:8000


# Template à installer
npx install prisma@6.19.0
npm instal hbs
ne pas oublier  de créer si nessesaire le fichier .env avec DATABASE_URL="file:./database.db" a l'intérieur
